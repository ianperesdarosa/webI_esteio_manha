O que é o HTML?

HTML significa "HyperText Markup Language", que em português é "Linguagem de Marcação de Hipertexto". É uma linguagem de marcação utilizada para criar a estrutura e o conteúdo de páginas da web. HTML é a base fundamental para a criação de sites e páginas na internet, permitindo que os desenvolvedores estruturem o conteúdo de maneira hierárquica e definam elementos como texto, imagens, links, formulários e outros tipos de mídia.

A linguagem utiliza uma sintaxe composta por "tags" (etiquetas) que são inseridas no código-fonte do documento HTML para indicar como o conteúdo deve ser formatado e exibido no navegador. Cada tag é composta por uma abertura, um conteúdo e um fechamento, sendo que a maioria delas é organizada em uma estrutura aninhada, criando uma árvore de elementos.

Aqui está um exemplo simples de código HTML que cria um parágrafo de texto:

<p>Isto é um parágrafo de exemplo.</p>

No exemplo acima, a tag `<p>` é usada para marcar o início de um parágrafo e a tag `</p>` marca o seu final.

Além de definir a estrutura e o conteúdo, o HTML também pode ser utilizado para incluir referências a outros recursos, como folhas de estilo CSS para estilização e scripts JavaScript para adicionar interatividade às páginas.

Em resumo, HTML é a linguagem básica para a construção de páginas da web, fornecendo a estrutura e os elementos necessários para criar documentos que podem ser visualizados em navegadores da web.

Principais tags HTML

Tags de texto

<p> - Representa um parágrafo

<span> - Representa um texto sem quebra de linha

<b> - Representa um texto em negrito

<i> - Representa um texto em itálico
